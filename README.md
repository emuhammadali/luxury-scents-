## 📋 Project Overview

**Luxury Scents** is a premium, fully responsive e-commerce website for a luxury perfume brand. This project demonstrates modern web development practices with a focus on user experience, interactivity, and visual appeal. Perfect for Final Year Project (FYP) or portfolio showcase.

### 🎯 Key Features

| Feature | Description |
|---------|-------------|
| 🛍️ **Shopping Cart** | Add/remove products, update quantities, calculate total |
| 🔍 **Product Search** | Real-time search functionality with overlay |
| 🏷️ **Filter System** | Filter products by category (Men/Women/Unisex) |
| 📊 **Sort Products** | Sort by price (low-high, high-low) and rating |
| ❤️ **Wishlist** | Save favorite products |
| ⏰ **Countdown Timer** | Dynamic promo timer for sales |
| 📱 **Responsive Design** | Works perfectly on all devices |
| 🎨 **Modern UI** | Glass morphism, gradients, smooth animations |
| 🖱️ **Custom Cursor** | Luxury feel with custom cursor effect |
| 📧 **Newsletter Signup** | Email subscription functionality |
| 📞 **Contact Form** | User inquiry submission |
